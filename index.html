Getting Started

Installation

Bower

$ bower install --save math-elements

Usage

<!doctype html>
<html>
  <head>
    <script src="bower_components/platform/platform.js"></script>
    <link rel="import" href="bower_components/math-elements/math.html">
  </head>
  <body>
    <math-elements></math-elements>
  </body>
</html>
